# Sound effects (optional)

The game plays **built-in synthesized sounds** by default — no files needed. But if you want a specific sound, drop an MP3 here with the exact filename and the game will use it instead:

| File | Plays when |
|------|------------|
| `buzz.mp3` | a captain buzzes in (the big one — heard on the PA) |
| `correct.mp3` | host marks an answer correct |
| `wrong.mp3` | host marks an answer wrong |
| `open.mp3` | buzzers open |
| `win.mp3` | winner screen |

Keep them short (`buzz.mp3` ideally under ~0.5s). The screen must have **"enable sound"** clicked once (browser rule) before any sound plays.
